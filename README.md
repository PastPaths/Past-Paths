# Past Paths Website

Official website for Past Paths.

## Structure

- `index.html` — Homepage
- `about.html` — About page
- `process.html` — Process page
- `experiences.html` — Experiences page
- `partners.html` — Partners page
- `contact.html` — Contact page
- `privacy.html` — Privacy policy
- `terms.html` — Terms and conditions
- `css/style.css` — Main stylesheet
- `images/` — Site images and brand assets
- `js/` — Site scripts

## Updating the site

When replacing files from an update ZIP, only replace the files listed in the update note. Do not delete unrelated pages, folders, images, scripts, or existing project files.

After replacing files, commit the changed files and push to `main`.
